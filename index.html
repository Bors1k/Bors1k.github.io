Разработка задания для WS "Инженерный дизайн CAD"

M1_M72_Project

Разработать полный комплект файлов для проведения первого модуля чемпионата по стандартам WS “Инженерный дизайн CAD”. Необходимо выбрать подходящую модель, разбить ее на подсборки и обозначить какие детали буду моделироваться участником, а какие участник будет получать готовыми. Выбрать детали и подсборки для создания чертежей, а также придумать сценарий для анимации и фотореалистичного изображения. Придумать текстовую часть задания и критерии проверки.
